# 多轮对话与指代消解测试集

> **目的：** 验证 RAG 系统的上下文记忆和指代消解能力
> **知识库：** IT 运维知识库
> **前提：** 在同一会话中连续提问

---

## 场景 1：VPN 问题深入排查

### 第 1 轮
**问题：** 我的 VPN 一直连接超时，怎么解决？

**预期回答要点：**
- 检查外网连接（访问百度测试）
- 确认服务器地址 vpn.company.com:443
- 尝试切换协议为 L2TP/IPSec
- 清除 VPN 客户端缓存
- 尝试备用服务器 vpn2.company.com

### 第 2 轮（指代消解 — 指代 VPN）
**问题：** 它的备用地址是什么？

**预期：** 系统理解"它"指 VPN，回答 vpn2.company.com。

### 第 3 轮（指代消解 — 指代协议）
**问题：** 你提到的那个协议怎么切换？

**预期：** 系统理解"那个协议"指 L2TP/IPSec，回答：设置 → 首选项 → 协议 → 改为 L2TP/IPSec。

### 第 4 轮（保持上下文）
**问题：** 如果还是不行呢？

**预期：** 系统理解讨论的是 VPN 连接超时问题，给出进一步建议：提交申告并附上当前 IP。

## 场景 2：打印机故障排查

### 第 1 轮
**问题：** 3 楼打印机卡纸了怎么办？

**预期回答要点：**
- 识别卡纸位置（进纸盒 / 硒鼓区域 / 出纸口）
- 轻拉纸张取出
- 检查纸张是否受潮
- 清空出纸槽残留碎片

### 第 2 轮（指代消解 — 指代打印机）
**问题：** 纸取出来了但还是提示卡纸，它的错误代码是 13.0000。

**预期：** 系统理解"它"指 3 楼打印机，并确认错误代码 13.0000 = 卡纸，建议检查是否有残余碎片。

### 第 3 轮（指代消解 — 指代纸张类型）
**问题：** 用什么样的纸比较好？

**预期：** 系统理解在打印机上下文中，回答：70-80g 标准 A4 复印纸，装纸前抖松。

## 场景 3：服务器告警处理

### 第 1 轮
**问题：** 收到一个磁盘 95% 的告警怎么办？

**预期回答要点：**
- SSH 登录服务器
- 清理 journal 日志、/tmp、Docker 旧镜像
- `du -sh /*` 定位大文件
- 若无法清理 → 提交扩容工单

### 第 2 轮（追问细节）
**问题：** 清理日志用哪个命令？

**预期：** 系统理解是上一轮的清理操作，回答：`journalctl --vacuum-size=500M`。

### 第 3 轮（指代消解 — 指代告警级别）
**问题：** 这种情况属于什么级别？

**预期：** 系统回答 P0 级（紧急），5 分钟内必须响应。

## 场景 4：跨知识域切换（验证上下文隔离）

### 第 1 轮
**问题：** 怎么添加网络打印机？

**预期回答要点：**
- Win+R → `\\print-server.company.com`
- 双击打印机名称自动安装

### 第 2 轮（切换话题）
**问题：** 信息安全方面，手机丢了应该怎么做？

**预期：** 系统能从信息安全规范库中找到内容，回答：Intune 远程锁定/擦除、联系 IT、修改密码。

### 第 3 轮（回到之前话题）
**问题：** 刚才说的添加打印机，如果找不到队列怎么办？

**预期：** 系统能回到打印机话题，帮助排查（检查网络、权限、联系 IT）。

## 场景 5：追问模糊化

### 第 1 轮
**问题：** 什么是 RPO 和 RTO？

**预期：** 系统从数据库备份手册中找到：RPO ≤ 1 小时，RTO ≤ 4 小时。

### 第 2 轮（模糊追问）
**问题：** 那个能再好一点吗？

**预期：** 系统可能不理解"那个"指什么。应追问或给出模糊回答。**这是典型的指代消解失败场景。**

### 第 3 轮（精确追问）
**问题：** RTO 能缩短到 1 小时内吗？

**预期：** 系统理解现在 RTO 是 4 小时，询问是否可缩短。知识库中可能无此答案，属低置信度。

## 场景 6：连续否定修正

### 第 1 轮
**问题：** 打印机是彩色的吗？

**预期：** 系统理解上下文不明确，可能反问哪一层/哪个打印机。

### 第 2 轮
**问题：** 我说的是 3 楼的那台。

**预期：** 系统回答：3F 东侧是 HP LaserJet M404（黑白），4F 西侧才有彩色打印机 HP M454。

### 第 3 轮
**问题：** 不对，我要的是行政部那台。

**预期：** 系统纠正回答：行政部是 HP LaserJet MFP M428（黑白+扫描+复印多功能一体机）。
