# 跨知识库检索测试集

> **目的：** 验证当用户同时选择多个知识库时，RAG 能否从正确的知识库中检索
> **默认场景：** 同时勾选「IT 运维知识库」和「信息安全规范库」

---

## 一、IT 运维 → 信息安全 交叉问题

### Q-C1
**问题：** VPN 连接时需要注意哪些安全事项？

**预期：**
- 从 IT 运维知识库：VPN 使用场景、连接方式
- 从信息安全规范库：禁止借账号给他人、使用期间受安全策略监控、禁止访问非法网站
- 两库信息应融合回答

### Q-C2
**问题：** 打印机打印的机密文件怎么处理？

**预期：**
- 从信息安全规范库：机密文件须立即取走、使用后碎纸机销毁
- 从 IT 运维知识库：打印机型号对应的队列名

### Q-C3
**问题：** 邮件发不出去，而且附件里有客户的合同。

**预期：**
- 从 IT 运维知识库：Outlook 排查步骤、网页版测试
- 从信息安全规范库：合同是机密级、应使用加密邮件发送外部

---

## 二、仅 IT 运维知识库

### Q-C4
**问题：** 服务器宕机了，怎么远程重启？

**预期：**
- 从 IT 运维知识库：IPMI/iLO 远程重启、或启动 Failover 切换至备机
- 从信息安全规范库：无相关内容或相关性低

### Q-C5
**问题：** 数据库 pg_dump 备份命令怎么用？

**预期：**
- 从 IT 运维知识库：`pg_dump -U opsmind -d opsmind -Fc -f <path>`
- 从信息安全规范库：可能有数据备份策略的相关内容

---

## 三、仅信息安全规范库

### Q-C6
**问题：** 公司对员工使用个人网盘有什么规定？

**预期：**
- 从信息安全规范库：禁止通过个人微信/QQ/网盘传输公司文档
- 从 IT 运维知识库：无直接相关内容

### Q-C7
**问题：** 离职员工的 IT 账号和数据怎么处理？

**预期：**
- 从信息安全规范库：离职后 AD 账号禁用、邮箱保留 30 天、OneDrive 保留 30 天后清除
- 从 IT 运维知识库：邮件服务器运维手册中有离职邮箱处理流程

---

## 四、两个知识库都不覆盖

### Q-C8
**问题：** 公司 HR 系统的请假流程是什么？

**预期：**
- 低置信度。两个知识库都不覆盖 HR 系统。
- 应从新员工IT入职指南中找到 OA 系统地址（https://oa.company.com）作为部分参考

### Q-C9
**问题：** 财务报销的审批链是什么？

**预期：**
- 低置信度。非 IT 范畴。

---

## 五、仅用单一知识库时不应混入其他库内容

### Q-C10
**场景：** 用户仅选择「IT 运维知识库」
**问题：** 怎么设置屏幕自动锁定？

**预期：**
- 从 IT 运维知识库：无此内容或仅有间接内容（电源管理中的睡眠相关）
- 不应从信息安全规范库获取（因为用户未选择该库）
- 若 IT 运维知识库无相关内容 → 低置信度回答

### Q-C11
**场景：** 用户仅选择「IT 运维知识库」
**问题：** 公司有哪些数据安全等级？

**预期：**
- 低置信度。此内容在信息安全规范库中，不在 IT 运维知识库。
- 若用户确实未选择信息安全规范库 → 应提示当前知识库不包含此内容，建议切换知识库或扩大范围

---

## 六、知识库选择与切换

### Q-C12
**操作验证：** 用户能否在问答过程中切换知识库？
**预期：** 当前会话已选择的知识库用于检索，切换后新问题应用新选择。

### Q-C13
**操作验证：** 同时选择两个知识库时，检索结果数量是否正确？
**预期：** 两个知识库各返回 top_k/2 个结果（或按配置的检索策略），总数为 top_k。

### Q-C14
**操作验证：** 两个知识库的向量维度不一致时如何处理？
**预期：** 系统应分别检索各自向量空间，在 RRF 融合步骤合并排序，而非混合向量计算。
